# IT Park Clone (Django Project)

This project is a clone of the IT Park platform built using Django.  
The system allows users to register, login, create profiles and manage projects.

## Features

- User Registration 🔐
- User Login / Logout 🔓
- User Profile System 🧾
- Profile Editing 📝
- Project CRUD (Create, Read, Update, Delete)-CRUD📍
- Project Detail Page 👀
- Skills section 🤹‍♂️
- Profile Image Upload 🌆
- Social Links (GitHub, LinkedIn, YouTube, Instagram)🌍

## Technologies Used

- Python --> 3.10.11 📥
- Django --> 4.2.2 📥
- HTML --> 5
- CSS
- SQLite
- Django Templates
